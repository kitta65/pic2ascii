# pic2ascii

> [!warning]
> This project is still a work in progress!

pic2ascii converts a line art into ascii art.

## Usage

```sh
git clone https://github.com/kitta65/pic2ascii.git
cd pic2ascii
make default
./bin/main path/to/input/image.png path/to/output/image.png
```
